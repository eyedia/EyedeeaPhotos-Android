# Fire TV App Requirements

## Authentication

- The Fire TV app uses a QR code-based login system.
- The app generates a device code and displays it as a QR code.
- The user scans the QR code with their mobile device to authenticate.
- Once authenticated, the app receives a token to access the user's account.

## Host URL

- The app should load the `<host>/view` page by default.

## Settings Page

- The settings page should be accessible and navigable with a remote control.
- It should display:
    - App logo
    - User details
    - Logout button
